<h2>はじめに──ニトロゲナーゼ補因子生合成研究の枠組み</h2>
<h3>窒素固定の化学量論</h3>
<p>$$\text{N}_2 + 8\text{H}^+ + 8e^- + 16\text{ATP} \xrightarrow{\text{ニトロゲナーゼ}} 2\text{NH}_3 + \text{H}_2 + 16\text{ADP} + 16P_i$$</p>
<h3>Mo型ニトロゲナーゼの2成分タンパク質</h3>
<table>
<thead>
<tr>
<th>タンパク質</th>
<th>遺伝子</th>
<th>構造</th>
<th>役割</th>
</tr>
</thead>
<tbody>
<tr>
<td>MoFeタンパク（ジニトロゲナーゼ）</td>
<td>nifD, nifK</td>

<td>N₂を直接結合・還元</td>
</tr>
<tr>
<td>Feタンパク（ジニトロゲナーゼ還元酵素）</td>

</tr>
</tbody>
</table>
<h3>3種類の金属クラスター</h3>
<ul>
<li><strong>[4Fe-4S]クラスター</strong>: Feタンパクに1個（電子供与体）</li>
<li><strong>Pクラスター [8Fe-7S]</strong>: MoFeタンパクの各αβ半分に1個（電子中継）</li>
<li><strong>FeMo-co [7Fe-9S-C-Mo]</strong>: MoFeタンパクの活性部位（基質還元）</li>
</ul>
<h3>生合成タンパク質の3機能グループ</h3>
<ol>
<li><strong>足場タンパク質</strong>: NifU・NifB・NifQ・NifEN（クラスター骨格の組み立て）</li>
<li><strong>シャペロンタンパク質</strong>: NifX・NafY・NifM・NifW・NifZ（輸送・成熟化補助）</li>
<li><strong>基質提供酵素</strong>: NifS・NifB・NifV（補因子原料の提供）</li>
</ol>
<h3>Pクラスターとfemo-coの組み込み経路の違い</h3>
<ul>
<li>Pクラスター → <strong>in situ 組み立て</strong>（最終位置でそのまま成熟）</li>
<li>FeMo-co → <strong>外部組み立て後挿入</strong>（複数タンパク質が段階的に組み上げ→挿入）</li>
</ul>

<blockquote>
<p>📊 Fig. 1 参照（原論文）</p>
</blockquote>
<ul>

<h3>電子移動経路</h3>

<h3>FeMo-coの配位環境</h3>
<p>$$\text{FeMo-co} = [\text{7Fe-9S-C-Mo}] + R\text{-homocitrate}$$</p>
<ul>
<li>αCys275 → 末端FeをMoFeタンパクに固定</li>
<li>αHis442 → Mo原子に配位</li>
<li>Mo周辺: 親水性環境、水素結合ネットワーク</li>
</ul>

### 第4章: 単純な鉄硫黄クラスターの生合成──NifSとNifUの役割

<video controls width="100%" src="https://archive.org/download/paper-explain-chemrev-9b00489/paper_chemrev.9b00489_ch14.mp4"></video>

<details>
<summary>解説スライド（クリックで展開）</summary>

<h2>単純な鉄硫黄クラスターの生合成──NifSとNifUの役割</h2>
<h3>NifSの反応（システインデスルフラーゼ）</h3>
<p>$$\text{L-Cys} \xrightarrow{\text{NifS (PLP)}} \text{L-Ala} + \text{S}^0 \rightarrow [\text{Fe-S cluster sulfur}]$$</p>
<ul>
<li>PLP依存性酵素、ホモダイマー（約88 kDa）</li>
<li>L-システイン <strong>だけ</strong> に反応する厳格な基質特異性</li>
<li>IscS（一般的[Fe-S]生合成のホモログ）との違い：機能交換不可</li>
</ul>
<h3>NifUの3ドメイン構造</h3>
<table>
<thead>
<tr>
<th>ドメイン</th>
<th>機能</th>
</tr>
</thead>
<tbody>
<tr>
<td>N末端ドメイン（NifU-N）</td>
<td>一時的[2Fe-2S]担持→Feタンパク供給</td>
</tr>
<tr>
<td>IscU様ドメイン</td>
<td>NifBなどへの前駆体供給</td>
</tr>
<tr>
<td>C末端ドメイン</td>
<td>恒久的[4Fe-4S]保持</td>
</tr>
</tbody>
</table>
<h3>nifU/nifS欠損の影響</h3>
<ul>
<li>nifU欠損: Feタンパク活性が95%低下、MoFeタンパク活性75%低下</li>
<li>nifU・nifS同時欠損: Feタンパク活性消失、MoFeタンパク活性90%以上低下</li>
<li>FeMo-co添加でも活性化できない → Pクラスター前駆体も不足</li>
</ul>

<h2>Feタンパク質の成熟と成熟化因子の相互作用</h2>
<h3>Feタンパク質の3つの役割</h3>
<ol>
<li><strong>ジニトロゲナーゼ還元酵素</strong>: 成熟[4Fe-4S]含有Feタンパクがe⁻供与</li>
<li><strong>Pクラスター形成補助</strong>: MoFeタンパクとの複合体で機能</li>
<li><strong>FeMo-co生合成</strong>: NifENとの複合体で機能</li>
</ol>
<h3>NifMの機能（PPIase型プロリルイソメラーゼ）</h3>
<ul>
<li>Feタンパクの <strong>Pro258</strong> がNifMの基質残基（C末端近傍に位置）</li>
<li>K. pneumoniae: NifM欠損でFeタンパクの安定性と二量体化が損なわれる</li>
<li>Pro258→Ser置換でNifM非依存型のFeタンパクになる</li>
</ul>
